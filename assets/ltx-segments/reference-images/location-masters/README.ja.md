# Location Masters

このフォルダには、`character + environment + style-master` から生成したロケーション固定マスター画像を保存します。

- ここに置く画像は、最終ショット生成時の主要参照として使います。
- 例:
  - `midnight-memory-master-vocal-booth-v01.png`
  - `midnight-memory-master-rooftop-v01.png`
  - `midnight-memory-master-river-bridge-v01.png`
  - `midnight-memory-master-glass-city-v01.png`
